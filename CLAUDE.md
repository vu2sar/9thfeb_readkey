# 9thfeb_readkey

## Project Overview
A C# console application (.NET Framework 4.5.2) that demonstrates reading user input and using `StringBuilder`.

## Structure
- `9thfeb_readkey.sln` — Solution file
- `9thfeb_readkey/Program.cs` — Main entry point
- `9thfeb_readkey/9thfeb_readkey.csproj` — Project file (targets .NET 4.5.2)

## Build
```
msbuild 9thfeb_readkey.sln
```

## What the Program Does
1. Prompts the user to enter a number
2. Reads and echoes it back
3. Uses `StringBuilder` to build and print a greeting message
